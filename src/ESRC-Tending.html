ESRC Tending Checklist (v1.0.1)

1. Supplies:
- Small Secure Container
 - Core Probe Launcher I x 1
- Core Scanner Probe I x 8
- Hugs (fireworks, snowballs, etc.) [optional]

2. Be notified of an existing cache in system that needs tending by Allison.

3. Locate Cache using an existing bookmark or via the Bounce Method.

4. Check the cache ownership:
 - If it's not [SC0UT][1420.] then the cache will need to be relocated.

5. Check the cache name:
 - It should be "EvE-Scout Rescue Cache - Stranded in this wormhole? Request help in the EvE-Scout channel" If not, please relocate.

6. Verify that the cache has a password.
- Passwordless caches need to be relocated.

7. Make sure there is only one cache on grid and the cache is not on grid with any other object.

8. Open cache.
- Inventory contents and resupply if needed.

9. Record cache information:
- On Tender data entry form in Allison and verify info is correct.
