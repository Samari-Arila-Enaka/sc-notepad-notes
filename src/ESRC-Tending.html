ESRC Sowing Checklist (v1.0.0)

1. Supplies:
- Small Secure Container
 - Core Probe Launcher I x 1
- Core Scanner Probe I x 8
- Hugs (fireworks, snowballs, etc.) [optional]

2. Skills:
- Make sure "Anchoring I" is trained!

3. Be notified of no cache in system by Allison.
- Click "Sow New Cache"

4. Find a good location:
- If you can, reuse the existing bookmark.
- ONLY use the Bounce Method for a new location if.
1. The cache was destroyed, or
2. The bookmark doesn't exist
- If unsure, ping @esr-coordinator in esr-lounge.

5. Launch the container "For Self", and Anchor it.

6. Place the above listed supplies into the container.

7. Orbit container at 4000m and cloak.

8. Set the cache name.
- Copy from Allison.

9. Set New Password on container:
 - Copy the new password from Allison.

10. Record cache information (on Sower data entry form in Allison).
- Verify this info is correct.

11. Confirm container is anchored.
- This step is mandatory!

12. If there is not an existing bookmark:
- Add a bookmark to the ESRC New Bookmarks folder.
